#  Password Security Analyzer

##  Project Description
Password Security Analyzer is a simple Cyber Security web project that checks the strength of a password in real time. It helps users create strong and secure passwords by analyzing different security rules.

---

##  Features
- Real-time password strength checking
- Checks password length (8+ characters)
- Checks for numbers
- Checks for special characters
- Checks for uppercase letters
- Visual strength indicator bar
- Weak / Medium / Strong password status

---

##  Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
